# Korvist School Management Website

A premium one-page React + Vite landing website for a school management app.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Upload to GitHub

Upload all files in this project **except**:

- `node_modules`
- `dist`

## Deploy on Vercel

1. Push the files to your GitHub repository
2. Open Vercel
3. Import the repository
4. Click **Deploy**

Vercel should detect the project automatically as a Vite app.
